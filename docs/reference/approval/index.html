Approval#

Module for sampling approval preferences: preferences in which a voter either approves or disapprove each candidate.

euclidean(num_voters: int, num_candidates: int, radius: float = 0.5, space: EuclideanSpace = EuclideanSpace.UNIFORM, dimension: int = 2, seed: int = None) → list[set[int]][source]#

Generates approval votes according to the Euclidean model.

In the Euclidean model voters and candidates are assigned random positions in a Euclidean space. A voter then approves of the candidates that are within a certain radius. In other words, a voter approves of all the candidates that are with distance radius of their position.

Several Euclidean spaces can be considered. The possibilities are defined in the EuclideanSpace enumeration. You can also change the dimension with the parameter dimension.

A collection of num_voters vote is generated independently and identically following the process described above.

Parameters:

  • num_voters (int) – Number of Voters.

  • num_candidates (int) – Number of Candidates.

  • radius (float, default: 0.5) – Radius of approval.

  • space (EuclideanSpace, default: UNIFORM) – Type of space considered. Should be a constant defined in the EuclideanSpace.

  • dimension (int, default: 2) – Number of Dimensions.

  • seed (int) – Seed for numpy random number generator.

Returns:

Approval votes.

Return type:

list[set[int]]

impartial(num_voters: int, num_candidates: int, p: float, seed: int = None) → list[set][source]#

Generates approval votes from impartial culture.

Under the approval culture, when generating a single vote, each candidate has the same probability p of being approved. This models ensures that the average number of approved candidate per voter is p * num_candidate.

Parameters:

  • num_voters (int) – Number of Voters.

  • num_candidates (int) – Number of Candidates.

  • p (float) – Probability of approving of any given candidates.

  • seed (int) – Seed for numpy random number generator.

Returns:

Approval votes.

Return type:

list[set]

Raises:

ValueError – When p not in [0,1] interval.

truncated_urn(num_voters: int, num_candidates: int, alpha: float, p: float, seed: int = None) → list[set[int]][source]#

Generates approval votes from a truncated variant of Polya-Eggenberger urn culture.

Note that for a given number of voters, votes are not sampled independently.

Parameters:

  • num_voters (int) – Number of voters

  • num_candidates (int) – Number of candidates

  • alpha (float,) – The dispersion coefficient (alpha * m! copies of a vote are put back in the urn after a draw). Must be non-negative.

  • p (float,) – Ratio of approved candidates.

  • seed (int, default: None) – The seed for the random number generator.

Returns:

Approval votes

Return type:

list[set[int]]

Raises:

ValueError – When alpha is a negative number. When p not in [0,1] interval.
